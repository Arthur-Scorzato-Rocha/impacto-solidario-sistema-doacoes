📌 Plataforma Digital de Doações Solidárias
✅ Requisitos Funcionais

RF01 – O sistema deve permitir o cadastro de doadores.
RF02 – O sistema deve permitir o cadastro de beneficiários ou instituições sociais.
RF03 – O sistema deve permitir o login de usuários cadastrados.
RF04 – O sistema deve permitir o cadastro de recursos para doação (alimentos, roupas, materiais escolares).
RF05 – O sistema deve permitir editar dados cadastrados.
RF06 – O sistema deve permitir excluir registros.
RF07 – O sistema deve permitir visualizar lista de recursos disponíveis.
RF08 – O sistema deve permitir registrar uma doação.
RF09 – O sistema deve permitir acompanhar o status da doação (pendente, aceita, entregue).
RF10 – O sistema deve permitir visualizar histórico de doações realizadas.

✅ Requisitos Não Funcionais

RNF01 – O sistema deve possuir interface simples e intuitiva.
RNF02 – O sistema deve funcionar em dispositivos móveis e desktops (responsivo).
RNF03 – O tempo de resposta do sistema deve ser inferior a 3 segundos.
RNF04 – Os dados dos usuários devem ser armazenados com segurança.
RNF05 – O sistema deve estar disponível 24 horas por dia.
RNF06 – O sistema deve permitir manutenção e atualização facilitada.
RNF07 – O sistema deve utilizar banco de dados relacional.
