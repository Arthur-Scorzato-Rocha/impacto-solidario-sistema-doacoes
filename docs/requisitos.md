# Requisitos do Sistema

## Requisitos Funcionais

- O sistema deve permitir o cadastro de doadores.
- O sistema deve permitir o cadastro de beneficiários.
- O sistema deve permitir o cadastro de itens para doação.
- O sistema deve permitir registrar solicitações de ajuda.
- O sistema deve permitir listar todos os itens disponíveis para doação.
- O sistema deve permitir atualizar o status dos itens cadastrados.
- O sistema deve permitir excluir registros que não são mais necessários.
- O sistema deve permitir visualizar relatórios simples sobre as doações realizadas.

## Requisitos Não Funcionais

- O sistema deve possuir uma interface simples e de fácil utilização.
- O sistema deve garantir o armazenamento seguro das informações.
- O sistema deve apresentar tempo de resposta rápido nas operações.
- O sistema deve permitir fácil manutenção e evolução futura.
- O sistema deve funcionar em computadores com configurações básicas.
